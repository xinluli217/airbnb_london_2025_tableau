# London Airbnb Market Analysis
A data-driven analysis of spatial pricing patterns, occupancy dynamics, and host structure in the London Airbnb market.

## Project Overview

This project analyses the London Airbnb market using listing, calendar, and review datasets. 

The analysis focuses on:

- Spatial distribution of prices
- Seasonal demand patterns
- Market activity (occupancy trends)
- Host concentration (Top 10 hosts)

The project combines data cleaning in R and interactive visualisation in Tableau.

## Data Source

Data is obtained from the Inside Airbnb dataset（https://insideairbnb.com/get-the-data/）:

- listings.csv
- calendar.csv
- reviews.csv
- neighbourhoods.geojson
